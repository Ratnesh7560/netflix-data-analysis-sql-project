Netflix SQL Data Analysis
Overview

This project contains SQL queries used to analyze the Netflix Titles dataset.
The goal is to practice basic and intermediate SQL skills by exploring Netflix movies and TV shows.

Dataset

The dataset includes information about Netflix content such as:

Title
Type (Movie / TV Show)
Director
Country
Release Year
Rating
Duration
Date Added
SQL Concepts Used
SELECT, WHERE, ORDER BY
LIMIT, DISTINCT
COUNT, AVG, MIN, MAX
GROUP BY, HAVING
Subqueries
CASE statements
String functions (CONCAT, TRIM, LENGTH, UPPER)
Date functions
Analysis Examples
Count Movies vs TV Shows
Find top countries producing Netflix content
Identify most common ratings
Find top directors by number of titles
Calculate percentage of Movies in dataset
Tools
MySQL
