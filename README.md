Netflix SQL Data Analysis
Project Overview

This project contains SQL queries used to explore and analyze the Netflix Titles dataset.
The goal of this project is to practice and demonstrate SQL fundamentals and intermediate data analysis skills using a real-world dataset.

The queries analyze Netflix content such as Movies and TV Shows, release years, ratings, countries, directors, and content trends.

Dataset

The dataset used is the Netflix Titles dataset, which contains information about content available on Netflix.

Main columns include:

show_id
type (Movie / TV Show)
title
director
country
date_added
release_year
rating
duration
listed_in (genre)
description
